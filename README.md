## 🌟 这是什么？

我是计算机专业学生，对“**如何用 AI 更聪明地学习**”充满好奇。  
这个仓库是我所有实验的集合：

- 📝 把混乱的课堂笔记变成结构化摘要  
- ❓ 自动生成选择题/填空题来自测  
- 💡 用生活例子理解抽象概念（如熵、傅里叶变换）  
- 🗂️ 构建可搜索的个人知识库  
- 🤖 尝试本地运行开源模型（如 Llama 3, Qwen），保护隐私

> 🎯 **目标不是让 AI 替代学习，而是让它成为我的“超级学习伙伴”**。
